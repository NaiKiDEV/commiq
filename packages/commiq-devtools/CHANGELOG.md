# @naikidev/commiq-devtools

## 1.0.0

### Minor Changes

- [`3fbf90b`](https://github.com/NaiKiDEV/commiq/commit/3fbf90b6a15b19dfefd9019477c86963f13bddf0) Thanks [@NaiKiDEV](https://github.com/NaiKiDEV)! - Initial release of all commiq packages.

  - **commiq**: Core event-driven state management with sealed stores, command handlers, projections, and the event bus.
  - **commiq-react**: React bindings with `useProjection`, `useCommand`, and `CommiqProvider`.
  - **commiq-effects**: Effects plugin for adding side-effect behaviors to events.
  - **commiq-persist**: Persistence plugin for durable state across sessions.
  - **commiq-otel**: OpenTelemetry instrumentation plugin.
  - **commiq-devtools-core**: Framework-agnostic devtools engine.
  - **commiq-devtools**: React-based devtools panel UI.
